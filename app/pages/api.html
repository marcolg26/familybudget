API

Questa applicazione dispone delle seguenti API

Metodo API Descrizione
POST /api/auth/signup Registrazione di un nuovo utente
POST /api/auth/signin Login di un utente
GET /api/budget/ Spese dell'utente attuale
GET /api/budget_in/ Spese effettuate da altri utenti che coinvolgono l'utente attuale
GET /api/budget/:year Spese dell'utente attuale nell'anno year
GET /api/budget/:year/:month Spese dell'utente attuale nel mese month dell'anno year
GET /api/budget/:year/:month/:id Dettaglio della spesa id nel mese month dell'anno year
POST /api/budget/:year/:month Aggiunta di una spesa nel mese month dell'anno year
PUT /api/budget/:year/:month/:id Modifica della spesa id nel mese month e anno year
DELETE /api/budget/:year/:month/:id Eliminazione della spesa id nel mese month e anno year
GET /api/balance Visualizzazione riassunto dare/avere dell'utente
GET /api/balance/:id Visualizzazione del bilancio tra l'utente attuale e un utente avente un certo id
GET /api/budget/search?q=query Cerca la spesa che corrisponda alla stringa query
GET /api/budget/whoami Se autenticato, restituisce le informazioni sull'utente
GET /api/users/search?q=query Cerca l'utente che corrisponda alla stringa query
